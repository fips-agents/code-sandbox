# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the code sandbox, please report it through one of these channels:

- **GitHub Issues**: Use the [Escape Vector](https://github.com/fips-agents/code-sandbox/issues/new?template=escape_vector.yml) issue template
- **Email**: For sensitive findings that shouldn't be public, contact the maintainers directly

## Security Architecture

The sandbox uses 6 layers of defense-in-depth isolation. See `docs/pentest-report.md` for the full security assessment.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.5.x   | Yes       |
| < 0.5   | No        |
